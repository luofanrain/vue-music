# music
 #项目展示地址
   http://www.luofanrain.club/music/#/count
   
   BUG ：转换手机模式看此项目时，需手动刷新一次程序才能正常运行
   
	 
 #程序说明
 
      全播放：把歌曲全部添加进入播放列表
			
      多选+下载：如若要把想要的歌曲加入播放列表，许先点“多选”，选中所要的歌曲之后再点“下载” 就成功添加进入播放列表了
			
      列表：显示播放列表
			
      附：点击播放按钮为当前歌曲，点击歌曲名字进去歌曲详情页
